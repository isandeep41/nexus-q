An extensible quantum computing framework built on Qiskit for designing, optimizing, and executing scalable quantum algorithms.

## 🗺️ Project Roadmap
- [x] v0.1.0: Core circuit generation logic via Qiskit Runtime
- [x] v0.2.0: Qiskit 1.0 Support & CI Integration
- [ ] v0.3.0: Abstract multi-backend support (PennyLane integration)
- [ ] v0.4.0: Basic measurement error mitigation layer

## Prerequisites

- Python 3.9+
- [Qiskit](https://qiskit.org/)
