# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-06-13
### Added
- GitHub Actions CI workflow for automated testing.
- Test suite validating `qiskit` imports and experiment scripts.
### Changed
- Upgraded the Superposition and Entanglement experiments to support Qiskit 1.0 `StatevectorSampler` primitive.

## [0.1.1] - 2026-06-13
### Added
- Standard repository status badges (Build, License, Python, and Qiskit versions) to the `README.md`.

## [0.1.0] - 2026-06-13
### Added
- Initial boilerplate setup (`README.md`, `CONTRIBUTING.md`, `.gitignore`, `LICENSE`).
- IBM Quantum setup guide (`SETUP_IBM_QUANTUM.md`) and config template.
- Quantum Superposition and Entanglement boilerplate experiments.
